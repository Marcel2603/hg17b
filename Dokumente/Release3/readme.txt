In diesem Release wurden die Anmeldefunktionen erweitert. Für die Verschlüsselte Anmeldung der Veranstalter fehlt noch die Übertragung der Schlüssel. Diese wird auf das nächste Release verschoben.

Die App und der Server lassen sich mit gradle compilieren. Die build.gradle-Datein und ein gradle-Wrapper sind in den jeweiligen Ordnern vorhanden. Der Server kann mit java -jar Server.jar ausgeführt werden.
